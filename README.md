# Smol Menubar (on Tauri)

Experimental variant of Smol Menubar built on Tauri and SvelteKit.


## Run SvelteKit Dev Server

```bash
pnpm run dev
```

## Run Tauri Dev Server

```bash
pnpm tauri dev
```
